# 03 Risk Spikes

기술 가정을 실험으로 검증하는 짧은 탐색 작업.

## How to use

- PRD의 ASM-### 중 결과에 큰 영향을 주는 가정 → SPIKE-###로 승격.
- Spike는 시간 박싱 (예: 1~3일). 결과는 여기에 기록.
- 결과가 결정으로 굳어지면 `08_DECISION_REGISTER.md` 또는 ADR로 옮긴다.
