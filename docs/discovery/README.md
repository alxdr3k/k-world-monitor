# Discovery

Use this directory for ongoing exploration, raw Q&A, product research, and temporary analysis.

Discovery notes are not current implementation authority.

Rules:

- If discovery creates a question, add it to `07_QUESTIONS_REGISTER.md`.
- If discovery creates an accepted lightweight decision, add it to `08_DECISION_REGISTER.md`.
- If discovery creates a major architecture decision, add an ADR under `docs/adr/`.
- If discovery changes product scope, update `01_PRD.md`.
- If discovery changes roadmap / status, update `docs/04_IMPLEMENTATION_PLAN.md`.
- If discovery changes active current position, update `docs/context/current-state.md`.
