# Q-032: ContentDraft 4-format auto-generate phasing (v1+)

## Context

DEC-005에서 v0 turn-key publish scope를 blog_long 1개로 lock. 4-format reuse
anchor (ADR-0011) 는 schema-level lock 유지하되 activation 시점은 v1+ phasing
결정 필요.

## Discussion

format별 활성 비용 / 우선순위:

| Format | 활성 비용 | 우선순위 권고 | 사유 |
|---|---|---|---|
| blog_long | v0 활성 | v0 | Astro 사이트 source, cite footnote 컴포넌트 native |
| newsletter | 중 | v1.1 | blog_long composer 재사용 + email-safe HTML conversion + Substack export. TTS 불필요 |
| youtube_long | 高 | v1.2 | TTS(Q-031) + chart/visual pacing + chapter 분할 + YouTube Data API metadata 자동화 (또는 Q-033 manual upload) |
| shorts | 中 | v1.3 | 60초 script 압축 + 자막 + (TTS 또는 BGM only). vertical video 자체 호스팅은 R2 부담 |

phasing 권고: **newsletter → youtube_long → shorts**

대안 검토:
- shorts 먼저 (60초 짧음 → 빠른 발행) — but Thesis 압축 품질 검증 부담 大
- youtube_long 먼저 (TTS + chart 등 인프라 한꺼번에 풀기) — 1인 운영 시간
  박싱 부담
- newsletter 먼저 (blog_long 재사용 가장 가까움) — Substack manual 발행
  (DEC-005)을 newsletter format 활성 시 자동화로 전환

5번째 메타 카테고리 활성 phasing은 별도 topic — Q-022 → DEC-004 가 4
메타 카테고리로 축소했으며 v1+ 누적 dossier 기반 재승격은 본 Q 와 분리.
