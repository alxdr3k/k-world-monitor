## Context

리뷰에서 신설. **2026-05-11 사용자 결정: 개수 상한 폐기 — 1인 운영 부담
안에서 필요한 만큼 누적 가능**.

후보 (분포 균형 고려):

| 도메인 | risk_observer | opportunity_observer | neutral |
|---|---|---|---|
| macro_finance | IMF (downside risks), BIS | McKinsey/Bain (sector outlook), Goldman Sachs Research, productivity reports | FRED/ALFRED, BLS, BEA, OECD, World Bank |
| geopolitics_security | UCDP, ACLED, WEF Global Risks | (drawdown reports) | UN, IAEA, government white papers |

| trade_supply_chain | SCMP (regulatory shock) | shipping forecasts | WTO, UNCTAD |
| climate_environment | IPCC, NOAA | clean tech adoption reports | NOAA data |
| technology_cyber_ai | CISA (cyber threat) | OpenAI/Anthropic research, tech adoption surveys | NIST |
| digital_assets | FSB crypto framework | spot ETF inflow reports | CoinGecko, Glassnode |

분포 균형 강제 (v0 entry 72 source 기준):
- risk_observer ≤ 50% (≤ 36개 of 72)
- opportunity_observer ≥ 25% (≥ 18개 of 72)
- neutral ≥ 15% (≥ 11개 of 72)

  가능
- 운영 부담 안에서 누적 자유 (상한 없음). v0 entry 72 source (한국 24개
  + 글로벌 48개, 경제 22 + 정책 17 + 사회 18 + 대중문화 15).
- ADR-0019 source_perspective tag manual 입력 v0 / LLM 자동 분류 v2+
- **2026-05-11 reflow (DEC-004 + DEC-009)**: 8 카테고리 → 4 메타 카테고리 +
  경제 우선 분포 + 한국 소스 보강. 상세 source list 는 `docs/research/
  source-seed-list-2026-05.md` (research 형식, 이 repo `data/sources_seed.
  yaml` 로 commit — 외부 repo 의존성 없음)
- **2026-05-11 cap 폐기**: 기존 30~50 cap 사용자 결정으로 제거. ADR-0016

  enforcement 아님
