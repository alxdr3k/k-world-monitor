# Q-030: counterclaim multi-relation v1 도입 우선순위

## Context

R25 GPT 메타 재검토에서 9 edge type 권고 (v0 5 + v1 4 = 9). Claude 비판
리뷰에서 v0 5만 유지 + v1 4 추가 + v2+ 인과 추론 edge 신중 도입으로 단계화.

본 Q는 v1 4개의 도입 시점과 우선순위를 finalize.

## Discussion

v0 (ADR-0013 INV-0013-1, 5 edge type):
- SUPPORTS / CONTRADICTS / QUALIFIES / UPDATES / SUPERSEDES

v1 (ADR-0013 INV-0013-4, 4 edge type 추가):
- WEAKENS — A는 B의 강도를 약화시킨다 (CONTRADICTS보다 약함)
- STRENGTHENS — A는 B의 강도를 강화시킨다 (SUPPORTS보다 강함)
- MITIGATES — A는 B의 영향을 완화시킨다 (downside scenario에 적용)
- AMPLIFIES — A는 B의 영향을 증폭시킨다 (upside scenario에 적용)

후보:
- (a) v1 한 번에 4개 추가
- (b) v1 단계화 — WEAKENS/STRENGTHENS 먼저(thesis-relative challenge), v1.5
  MITIGATES/AMPLIFIES (impact_targets와 연결)
- (c) v2+ 검토로 미룸

권고: (a). 4개 edge type은 동시에 schema migration이 비용 동일. ADR-0019의
bidirectional framing 활성화 시점에 4개 모두 자연.

미도입 (v2+에서도 신중 검토):
- CAUSES — 인과 추론 edge, LLM hallucinate 위험
- INCREASES_RISK_OF / DECREASES_RISK_OF — 같은 위험
- NARROWS_SCOPE — QUALIFIES와 중첩, 별도 의미 불명확

## Resolution
