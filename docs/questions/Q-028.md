# Q-028: LLM API cost 통제 정책

## Context

ADR-0006 NFR-004 일일 LLM 비용 상한 정책을 정밀화. R24 자체 audit에서 신설.

ADR-0006 INV-0006-3 (prompt caching + batch API) + INV-0006-5 (run_ ledger
cost 기록) 보존.

## Discussion

권고:
1. **Anthropic prompt caching**: 시스템 prompt + extractor schema는 cached
   prefix. cache hit rate ≥ 70% 목표 (SPIKE-003 검증).
2. **batch API**: 비실시간 워크로드(claim extraction, cite check overclaim,
   thesis 압축)에 batch 적용. 실시간 워크로드(manual feedback policy_gate
   검증, intervention severity 산정)는 sync.
3. **Haiku 4.5 1차 + Sonnet 4.6 escalate**: ADR-0006 INV-0006-1 보존.
   reliability_tier ∈ {medium, low} OR extraction_confidence < 0.85 escalate.
4. **per-day spend ceiling**: 일일 USD $5 (예시, 운영 중 조정). run_ledger
   에서 daily SUM(cost) 비교, 임계 도달 시 큐 throttling + alert.
5. **per-week ceiling**: $25 (5 × 5). 주간 합산도 모니터링.
6. **alert webhook**: metrics_alerts 테이블에 임계 초과 row insert. v0 stdout
   로깅 + v1 webhook.

cost 모델 예시 (Haiku 4.5 기준, 2026-05):
- claim extraction 평균 토큰: 입력 1000 / 출력 200 → ~$0.0008/claim
- 일일 1000 claim extraction → ~$0.80/day
- Sonnet escalate 10% 가정 → +$0.20/day
- 일일 cite check overclaim 100건 → ~$0.10/day
- 총 ~$1.10/day 베이스라인, $5 상한은 5배 여유

## Resolution

**Resolved by DEC-008 → re-resolved by DEC-010** (2026-05-11) — DEC-008
의 Anthropic-only 3-tier (Haiku/Sonnet/Opus) 가 사용자 결정으로 supersede
됨. DEC-010 은 multi-vendor 4-tier 로 재구성:

- **default vendor = OpenAI (GPT 계열)**, Anthropic dual-vendor (Sonnet/
  Opus) 는 도메인 우위 override + cross-vendor review 용도, Google
  (Gemini) 은 탐색 보조 + Tier 3 비용효율 우위 시만 (메인/리뷰 X).
- **Tier 매핑 기준 = 성능/분야/특성** (가격 단독 매핑 금지). Tier 0
  (frontier reasoning) = GPT-5.5 Pro extended thinking + Claude Opus 4.7
  xhigh effort 동 tier.
- **Cross-vendor review 강제 단계 3 종**: publication preflight cite check
  overclaim / scenario validate (3') adversarial pass / 운영자 명시
  high-stakes thesis. 그 외 단계 cross-vendor 의무 X.
- **Data Science Module (ADR-0024)** 통과 의무 — 1000+ rows / 50KB+
  raw dataset 은 derived metric 으로 압축 후 LLM 입력.
- **Cost ceiling**: soft $5 / hard $7.5 / weekly $25 그대로 + Tier 0
  일일 cap 5회 추가 + backfill bucket. baseline 재추정 $1.20~1.45/day.
- **KPI 6개**: DEC-008 KPI 5 + cross_vendor_review_coverage ≥ 0.95 추가.

ADR-0006 (Haiku 1차 + Sonnet escalate) 도 ADR-0023 (LLM routing v2) 로
supersede 됨.
