# Q-022: v0 카테고리 8개 + tag 5개 + axis transmission_channel

## Context

ideation Round 22에서 카테고리 확장 논의. GPT v0 7개 + secondary 1
(digital_assets) + tag 5개 + axis 6개 권고 → Claude 비판 리뷰로 axis 4개로
축소 (confidence_basis와 content_role은 기존 ADR-0005/0007과 중첩).

Round 22 sub-Q (비트코인): digital_assets는 secondary category 추가. bitcoin은
entity/signal node, stablecoin과 분리.

## Discussion

v0 카테고리 8개:
- **core 7**: macro_finance / geopolitics_security / health_biosecurity /
  energy_commodities / trade_supply_chain / climate_environment /
  technology_cyber_ai
- **secondary 1**: digital_assets (bitcoin entity + stablecoin separate)

tag 5개 (dossier 후보로 시작, 누적 시 카테고리 승격):
- social_stability_information
- demographics_migration
- food_water_security
- governance_institutions
- critical_infrastructure

cross-cutting axis 4개:
- region (global / US / China / EU / Korea / Japan / Middle East / ASEAN /
  Russia / Sub-Saharan / LatAm)
- time_horizon (Q-001 → tactical/near/mid/generational 4-tier)
- shock_type (demand/supply/financial/political/biological/climate/technological/social
  + v2+ 양방향화)
- transmission_channel (energy / credit / fx / policy / technology /
  information / population)

폐기 axis 후보:
- confidence_basis (ADR-0005 reliability_tier + extraction_confidence + claim_
  status와 중첩 — 흡수)
- content_role (ADR-0009 scenario_assumption.role + scenario.counterclaim_ids
  + scenario.falsifiers + scenario.monitoring_signals와 중첩 — 흡수)

## Resolution

**Resolved by DEC-004** (2026-05-11) — v0 turn-key 시간 박싱을 위해 카테고리
8개에서 4 메타 카테고리(정책 / 경제 / 사회 / 대중문화)로 축소. 기존 8 카테고리는
4 카테고리 안의 `subtopic_tags[]` 로 강등 보존 — v1+에 누적 dossier 기반
재승격 경로 유지 (Q-032 phasing). cross-cutting axis 4 (region / time_horizon
/ shock_type / transmission_channel) 는 그대로 유지. enum yaml은 이 repo
`data/categories.yaml` v0 seed 시점 commit (외부 repo 의존성 없음).
