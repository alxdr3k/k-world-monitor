# DEC-011: v0 turn-key 첫 발행 sub-topic lock — 한국 부동산 폭락 시나리오

## Context

Q-036 은 DEC-009 의 상위 주제 "한국 부동산 시장/정책의 현주소, 리스크" 의
sub-topic 좁히기를 PUB-1A.5 진입 시점 사용자 결정으로 분리. 사용자 발화
(2026-05-11):

> 높은 가계부채 비율, 최근 몇년간 청년층에 집중되어온 영끌, 청약, 특별
> 공급, pf 부실 등이 방향이 잘못된 금리정책 위에서 현 정부의 강력한 부동산
> 대책 정책들을 연달아 마주했을 때 펼쳐질 수 있는 그림. 부동산 폭락

→ Q-036 5종 후보 중 단일 선택 아니고, **다중 axis 결합 cautionary 시나리오**.

## Decision

v0 turn-key MVP gate (PUB-1A.5) 의 첫 발행 sub-topic 을 다음으로 lock:

### Sub-topic

> **한국 부동산 폭락 시나리오: 누적 risk factor × 방향이 잘못된 금리정책
> × 현 정부 강력한 부동산 대책 결합 시 펼쳐질 수 있는 그림**

### 구성 axis (5 개 결합)

1. **누적 risk factor**:
   - 높은 가계부채 비율 (한국 GDP 대비 가계부채 ~100%+)
   - 최근 몇 년 청년층 영끌 매수 (2020~2022 영끌 코호트)
   - 청약 / 특별공급 의 제도적 distortion (가격 신호 왜곡)
   - PF (프로젝트 파이낸싱) 부실 (건설사 신용 risk + 금융기관 노출)
2. **trigger 1 — 잘못된 금리정책**: 한은 vs 시장 mismatch / Fed 발 글로벌
   금리 환경 / 한국 통화정책 lag
3. **trigger 2 — 강력한 부동산 대책**: DSR / LTV / 양도세 / 청년 특별공급
   등 연달은 정책 변화
4. **outcome scenario**: 부동산 폭락 (cautionary thesis)
5. **horizon**: 1~3년

### Thesis 권고 (운영자 진입 시 최종 결정)

- stance: `cautionary` (ADR-0019 INV-0019-1 enum)
- market_stance: `bearish` (한국 부동산 자산 가격) 또는 `volatility_up`
- bidirectional_weight_intent: `risk_observer` (ADR-0025 INV-0025-3 enum)
- **단 polarity-symmetric counterclaim 강제** (ADR-0009 INV-0009-1) —
  "왜 폭락 안 할 수 있는가" opposing evidence 최소 2개:
  - 정부 개입 (PF 부실 막기 / 한은 금리 인하 / 임대주택 공급)
  - 인구 감소·도시화 dynamics 의 장기 region 차등 (수도권 vs 비수도권
    decoupling)
  - 글로벌 부동산 cycle 과의 비대칭 (한국 = 특수 정책 시장)

### Target 비대칭 분석 (REQ-021 / ADR-0019)

`impact_targets[]` + `impact_direction_by_target` (dict):

| target | impact_direction | 근거 |
|---|---|---|
| 청년 영끌 코호트 (2020~2022 매수) | **downside (severe)** | LTV 한계 + 금리 부담 + 매도 출구 좁아짐 |
| 1주택 자가 거주자 | downside (moderate) | 자산가치 ↓ 단 거주 안정성 유지 |
| 다주택자 | downside (severe) | 양도세 + 보유세 + 자금 부담 동시 |
| 임차인 (전세) | mixed | 전세가율 변동 / 갭투자 출구 막힘 → 전세 안정 또는 보증금 회수 risk |
| 임차인 (월세) | upside (mild) | 매매가 ↓ → 월세 시장 안정화 압력 |
| 잠재 매수자 (대기) | upside (mild) | 매수 기회 (단 timing risk) |
| 건설사 (중소) | downside (severe) | PF 부실 직격 |
| 건설사 (대형) | downside (moderate) | 시장점유율 ↑ vs 신규 분양 부진 |
| 금융기관 (PF 노출 대형) | downside (severe) | 부실 자산 비율 ↑ |
| 정부 / 한은 | mixed | 개입 비용 vs 정책 신뢰도 |

### Transmission channels (REQ-021)

- `credit` (가계부채 → 가처분 소득 → 소비)
- `policy` (정책 변화 → 가격 신호 → 매수 timing)
- `fx` (KRW 약세 시 외국인 매수 ↑ / 약세 시 외국인 매도 ↑)
- `information` (정책 발표 timing + 청약 시장 신호)
- `population` (인구구조 + 청년층 자산형성 cohort)

### Narrative arc 선택 (PUB-1A.5 진입 시 운영자 결정)

DEC-011 의 5 axis 결합 cautionary 시나리오는 단일 axis 보다 **더 깊이 있는
인사이트가 가능**하다 — 각 axis 가 다른 axis 의 evidence 가 되어 층층이
근거를 형성한다. 예:

- 높은 가계부채 (axis 1) → 청년 영끌 LTV 70% (axis 2 evidence) → PF 부실
  cascade (axis 3 amplifying) → 잘못된 금리정책 timing (axis 4 trigger) →
  강력한 정부 대책 시점 충돌 (axis 5 trigger 2)
- 단일 axis (예 가계부채만) 분석은 누적 cascade 누락 — 5 axis 결합 시에야
  "왜 지금 시점인가" 의 multi-causal trigger 명확

단 blog_long 1 편 분량 (5~10k 단어 적정) 과 5 axis 깊이의 trade-off 존재.
**PUB-1A.5 진입 시 운영자가 narrative arc 옵션 선택** + EditorialIntent
`alignment_criteria` 에 명시:

- **옵션 (A) — "전체 그림" series 진입글**: 5 axis cascade 누적 effect 위주
  본문, 각 axis 1~2 단락 evidence + monitoring signal. blog_long 5~7k 단어.
  후속 글 series (PUB-1B.x) 가 각 axis 또는 axis 결합 deep-dive.
- **옵션 (B) — "deep-dive" 단일 long-form**: 5 axis 모두 본문 깊이 (각 axis
  의 transmission channel + target 별 비대칭 + 반대 증거 + monitoring).
  blog_long 분량 확장 (15~20k 단어, long-form essay 형식).
- **옵션 (C) — "intersection" 단일 글**: 5 axis 교차점 중 highest-impact
  combination (예 "청년 영끌 코호트 × PF 부실 × 잘못된 금리 timing 의 동시
  발화") 만 본문 + 나머지 axis 는 appendix / "확장 분석 예고" 로 후속 글
  trigger. blog_long 6~8k 단어.

권고: **(A) 전체 그림 series 진입글** — DEC-011 의 5 axis backbone 을
첫 글이 manifesto / index 로 명시 + 후속 글 series 가 각 axis 또는 axis
결합 deep-dive. 운영자 결정 시 (B) / (C) 도 가능. **5 axis 모두 본문
포함 의무 (사용자 결정 lock) — 분량 trade-off 는 narrative arc 선택의
trade-off 일 뿐, axis 축소 사유 아님**.

## Rationale

- **다중 axis 결합 시나리오**: Q-036 단일 후보로는 사용자 인사이트 (5개 risk
  factor 의 동시 발화) 압축 불가. cautionary thesis 가 transmission channel
  + target 비대칭 + monitoring signal 로 분해 가능 — ADR-0019 bidirectional
  framing 강점 최대 활용
- **운영자 mental model 정합**: 한국 macro / 부동산 / 자산형성기 1인
  직장인 / 청년 영끌 cohort 는 운영자 + audience 가 모두 직접 stake 가진
  주제 — content quality 의 thesis 판단력 강함 (GPT 메타 리뷰 1순위 병목
  = 운영자 thesis 판단)
- **차별화**: 글로벌 macro content 풍부 vs 한국 특수 부동산 시장 + 청년
  세대 cohort 비대칭 분석 희소 — GPT 메타 리뷰 #5 잠재력 (한국 관점 차별
  화) 자연 정합
- **source 정합**: 한국 부동산 specific source (KAB / MOLIT / HF) 누락은
  Q-021 후속 등록 의무. v0 첫 발행 직전 등록 trigger
- **polarity-symmetric counterclaim 강제 (ADR-0009)**: cautionary thesis
  지만 opposing evidence 최소 2개 (정부 개입 + 인구 decoupling + 한국
  특수성) 의무. balance metric (ADR-0019 REQ-022) 의 기계적 균형이 아니라
  "반대 증거를 알고도 왜 cautionary thesis 를 채택했는지" 설명 의무
